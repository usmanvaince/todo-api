# todo-frontend

## Project setup
```
1. Clone the repo from https://github.com/usmanvaince/todo-api.git
2. composer install
3. cp .env.example .env
4. Set the Database credentials
5. php artisan key:generate
6. Run migrations
8. If you don't have email credentials for sending email, you can comment out the code in Register Controller and get the verification code from database.
